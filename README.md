## 🧭 What this is

open-swe is an open-source coding agent for Windows. It helps you work with code by handling tasks in the background while you keep using your PC.

Use it when you want to:
- ask an AI to help with coding tasks
- run long tasks without keeping a window open
- connect with common AI models from OpenAI or Anthropic
- manage software work in a simple desktop setup

## ⚙️ Basic use

A simple way to use open-swe:
1. Enter a task in plain English

- check your internet connection

### The app opens, but nothing happens
- close the app and open it again

- test with a short prompt
- restart Windows if needed
